Instalación de TravelShare

Creadores: Brais Bértolo Senra, Omar Razzouki Rodríguez

Servidor

Se utilizó XAMPP. Para ello, accedemos como administradores y activamos el panel de control de Apache y de MySQL.

Base de Datos

La base de datos está dentro de la carpeta "Documentos extra", se llama portal_viajes y es un archivo en formato sql. Una vez localizada, accederemos desde el navegador a localhost/phpmyadmin.

Una vez dentro, veremos que en el menú superior aparece la opción "Importar", a la que accederemos y dentro de ella en el apartado de "Fichero a Importar" seleccionaremos el fichero con la base de datos. Para finalizar la importación, bajaremos hasta la parte de abajo y seleccionaremos "Importar".

Todo debería de haberse importado de manera correcta.

Iniciar la Aplicación

Se deberá copiar la carpeta TravelShare.

Después, buscaremos la carpeta "XAMPP" creada tras instalar la aplicación. Dentro de esa, encontraremos otra carpeta llamada htdocs. Entramos y copiaremos nuestra carpeta ahí.

Para acceder a la aplicación, buscaremos en el navegador localhost/TravelShare lo que nos llevará al index.

Descripción de Roles

  • Administrador: Es el encargado de mantener el orden en la plataforma. Tiene la capacidad de validar las crónicas subidas por los usuarios antes de que sean públicas, así como de eliminar las publicaciones si pasaron el filtro de manera erronea. Puede acceder a todas las opciones de la aplicación
  • Explorador: Es el alma de TravelShare. Este rol identifica a aquellos usuarios activos que desean compartir sus experiencias de viaje, subir fotos y mostrar sus rutas para inspirar a los demás usuarios.
  • Normal: Un rol pensado para aquellos que disfrutan consumiendo contenido. Pueden navegar, y guardar sus crónicas favoritas.

Comentarios adicionales

  • El index nos permite ver la lista de crónicas publicadas, cualquier interactuación con la página nos redirigirá al inicio de sesión.
  • La página no tiene un botón de retroceder, está hecha para que para retroceder, por ejemplo entrando en una crónica desde la sección explorar, se pulse en la sección del menú lateral para volver hacia atrás.
  • En la sección donde el administrador puede ver las crónicas pendientes, aprobadas y rechazadas, las crónicas rechazadas y aceptadas serán solo las de los últimos 30 días.
