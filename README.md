# TravelShare

Proyecto de red social de viajes.

Este README sirve para dejar claro qué es el proyecto y cómo arrancarlo rápidamente. [web:31]

## Importante
El proyecto no está en su fase final como tal, muchas secciones están duplicadas por rol (por ejemplo `admin-explorar.php`, `usernormal-explorar.php`, etc.) y normalmente se resuelve con redirecciones según el usuario. Esto hace que se pierda parte de la gracia de usar PHP “bien” (un único archivo por sección y renderizado condicional).

## Antes de empezar
Lee primero la documentación de instalación:

`TravelShare/Documentos extra/instalar-aplicacion.html`
